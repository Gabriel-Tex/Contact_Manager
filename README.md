# Contact Manager
Gerenciador de contatos com CRUD em C++.

## Notas de aprendizado
Notas referentes aos conceitos e features da linguagem C++ nos quais me aprofundei durante o desenvolvimento do projeto.

* [Strings](notes/string.md)
    * std::string
    * std::getline()
    * std::ws
    * length() e ssize()
    * string_view
* [Compound Data Types](notes/compoundDataTypes.md)
    * Tipos de dados compostos
    * Referências
    * lvalues e rvalues (categorias de valor)
    * no-const e const lvalue reference
    * ponteiros
* [Maps](notes/map.md)

OBS.: nem todos os assuntos e recursos das notas foram/serão utilizados no desenvolvimento do projeto. As anotações são baseadas num estudo geral de cada tópico realizado antes do desenvolvimento, que geralmente são capítulos compreendidos em https://www.learncpp.com.
